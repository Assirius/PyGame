Управление up/down/left/right - все через стрелочки
Закрыть игры крестик слева вверху.
Преждевременное закрытие игры до Game over - сохраняет прогресс, имя игрока и счет, после следующего открытия сможете начать с места закрытия.

После Game Over - для продолжения игры есть 3 сценария:
1.Закрыть приложение нажав на крестик и перезапустить игру.
2. Нажать клавишу Enter - перекинет в меню выбора имени игрока, после заполнения можно продолжить игру.
3. Нажать клавишу Space(пробел) - возобновление игры под тем же игровым именем.
